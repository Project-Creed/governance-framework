# Governance Book Index

This index organizes the manuscript source in `chapters/` into a recommended reading sequence.

## Recommended Reading Order

1. [Introduction](../chapters/00-introduction.md)
2. Read Part I through Part IV in numeric order.
3. Revisit chapters based on the governance domain you are facing now: citizenship, representation, public trust, lawmaking, policy, taxation, budgeting, administration, transparency, corruption, elections, rights, public goods, regulation, emergencies, federal scale, foreign policy, or technology.
